破墙之刃 自建$3.99每年的独享Shadowsocks科学上网服务器

2015-02-01

本文讲述了如何使用价格低廉的Bandwagonhost(搬瓦工)VPS建立属于自己独享的Shadowsocks科学上网服务器。

别以为建一堵墙就可以永远的把危险挡在外面，巨人渴望进来，我们渴望出去，只要越过这道墙，便没有什么不再可能！

  版瓦工VPS现在后台自带一键安装SS了哟~所以这个教程的后面的安装部分并没有什么卵用了。。

  随着GFW（国家防火墙）的升级，很多VPN服务都失效了，而且剩余的VPN服务的价格也是水涨船高。今天我们来使用Bandwagonhost(搬瓦工)提供的低廉的VPS服务来建立我们自己独享的科学上网服务器。

  我们的方法是在境外的VPS上部署Shadowsocks服务来为我们提供正确的科学上网姿势。

1、Shadowsocks是一款轻量级的隧道代理，工作稳定、安全。

2、Bandwagonhost是美国IT7网络公司旗下的美国VPS服务商,价格便宜性能强劲,国内访问速度快。

3、我们这里使用的VPS的配置是 单核/64MB内存/100GB每月流量 足够一个人的使用。

###1 购买和配置 Bandwagonhost VPS服务器
1.1 打开 Bandwagonhost 网站；

1.2 点击 Register 进行注册，如图所示，填写好内容后，登录；

1.3 点击 购买链接 进入选购VPS页面；

1.4 选择如图所示的配置 Micro-64

1.5 VPS的选项如图所示，选择 西海岸-亚利桑那州 （凤凰城的速度比较快）

1.6 点击加入购物车，并进行支付。搬瓦工使用的PalPay，可以使用银联银行卡或者信用卡付款；

1.7 付款完毕之后点击 My Service ，进入VPS列表；

1.8 点击如图所示的按钮，进入VPS管理面板；

1.9 点击stop，关闭VPS；

1.10 点击 Install New OS，并选择debian系统（低配置VPS必备）；

1.11 等待片刻，获得SSH端口和root密码；
