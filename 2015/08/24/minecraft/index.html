仓鼠的MC误国服3.5周目

2015-08-24

永夜误国群专用服务器

####服务器信息：

  • 版本：1.7.10 with Forge
  • 人数：20
  • 线路：贵州电信
  • 类型：独立服务器

####主要MODs：

主题类：[抖M向]

  • 虚无世界
  • 神圣RPG

冒险类：

  • 战斗高塔
  • 沉船

装备类

  • 最终的剑
  • 双持
  • 更好的附魔
  • 属性装备
  • 酱魂

其余还有好多辅助类的就不一一说明了

####重要信息
1、新版本的服务器的注册/登录方式有所改变，请仔细阅读以下说明！

2、新版本的MOD改变了很多东西，请务必阅读主要MOD教程！！！

####安装：

#####1、下载
（1）Java Runtime Environment 8.0

  • 32位 （戳我下载）
  • 64位 （戳我下载）

（2）Minecraft 客户端

本体：(本体版本 Ver 3.5.0)

本周目服务器支持自定义史蒂夫MOD，该MOD不支持所有32位系统以及XP系统，请不符合安装条件的切勿下载自定义史蒂夫补充包。
  • 下载低配/标准版本
  • 下载高配版本
  • 下载自定义史蒂夫补充包

v3.5.1补丁包

忘记了一个家具mod，请自行下载丢到.minecraft/mods文件夹中

  • 下载v3.5.1补丁包

#####2、安装步骤

  • 2.1 打开1-（1）中下载的 Java Runtime Environment 8.0，一路下一步到底即可完成安装；

  • 2.2 解压缩下载下来的压缩包；

  • 2.3 打开仓鼠误国服文件夹；

  • 2.4 打开mclauncher.exe；

  • 2.5 请填写游戏名字！

  • 2.6 最大内存32位系统请填1024，64位系统可填写电脑内存的一半（如果不能正常运行可以重复2.4~2.6步骤，并将最大内存改为1024）

  • 2.7 java路径在安装完 Java Runtime Environment 8.0会默认出现，如果你还没有安装可随便指定一个然后关闭，再 重复步骤2.1-2.7

#####3、首次进入游戏

  • 3.1 请务必选择版本 最后一项；

  • 3.2 进入游戏之后，点击进入服务器；

  • 3.3 进入服务器之后，请按下键盘T；

  • 3.4 这个步骤为注册账号，请输入/register 密码 重复密码；（请自行将密码、重复密码替换为自己的密码，响应超时时间较短，请输入的时候快
    点，或者直接ctrl+c、ctrl+v）

  • 3.5 至此，就可以享受误国的乐趣；

#####4、非首次进入游戏

  • 4.1 请务必选择版本 最后一项；

  • 4.2 进入游戏之后，点击进入服务器；

  • 4.3 进入服务器之后，请按下键盘T；

  • 4.4 这个步骤为登录账号，请输入/l 密码；(这里的密码是你在3.4中注册的密码)

#####5、常用命令

  • /sethome 设置该点为 家
  • /home 回 家
  • /back 返回传送之前的位置
  • /tpa 玩家名字 传送到某玩家
